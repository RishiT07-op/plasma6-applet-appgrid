## 🔧 Troubleshooting Common Issues

- **App does not start**  
  Make sure KDE Plasma 6 and required dependencies are installed. Check for error messages in the terminal if you run it from there.

- **Launcher not showing apps**  
  The app scans your installed software list. Run a system update to refresh app listings, then restart the launcher.

- **Installation errors**  
  Verify you downloaded the correct package for your Linux distribution and version. Use your package manager to resolve any missing dependencies.

- **Graphics issues**  
  If the grid appears blank or distorted, check your GPU driver and Plasma compositor settings.
